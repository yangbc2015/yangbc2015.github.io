# 背景音乐说明

## 如使用背景音乐功能

1. 将您的音乐文件重命名为 `bg-music.mp3`
2. 放入此目录 (`static/music/`)
3. 重新构建网站

## 免费替代方案

如果您需要免费的背景音乐，可以考虑以下建议：

1. **免费音乐库**：
   - [Free Music Archive](https://freemusicarchive.org/)
   - [Incompetech](https://incompetech.com/)
   - [Musopen](https://musopen.org/)

2. **免费版权音乐 (CC0/CC-BY)**：
   - 古典音乐（已过版权保护期）
   - 独立艺术家作品

3. **自制音乐**：
   - 使用 AI 音乐生成工具（如 Suno AI《波点音乐等）

## 技术说明

- 支持格式：MP3 (`.mp3`)
- 建议音量：已设置为 30%，避免影响阅读
- 播放模式：循环播放
- 自动播放：现代浏览器限制自动播放，需要用户点击页面后才能开始播放
