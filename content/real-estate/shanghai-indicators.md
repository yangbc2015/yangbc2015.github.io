# 上海房地产核心指标监控体系

> 建立三层监控体系：宏观层决定3年趋势，中观层决定6个月趋势，微观层决定1个月波动
> 
> **核心15项指标**：库存去化周期、成交价、库存量、法拍房数量/数据、投行研报、银行信贷额度变化、中介信心指数、居民杠杆率、收入/杠杆比、供需关系、通胀预期、冰山数据、M2指数、挂牌量数据

---

## 一、核心指标速查表（15项必看）

### 📍 市场交易指标（5项）

| 指标 | 为什么重要 | 数据获取渠道 | 更新频率 |
|------|-----------|-------------|---------|
| **成交价** | 市场真实价格水平；关注网签价与挂牌价的背离程度 | 房管局网签数据、链家成交价、冰山指数 | 日/周 |
| **库存去化周期** | <6个月上涨动力，>18个月下跌压力；最关键供需指标 | 克而瑞、易居研究院、链家研究院 | 月度 |
| **库存量**（新房+二手房） | 供给端压力的直接体现；上海二手房挂牌>20万套=压力巨大 | 网上房地产（新房）、冰山指数（二手）、链家APP | 日/周 |
| **供需关系** | 供需比>1.5:1为买方市场，<1:1为卖方市场 | 链家成交量/新增挂牌量比值、克而瑞供需比 | 周度 |
| **二手房议价空间**<br>（挂牌价-成交价）/挂牌价 | **市场温度指标**<br><3%卖方市场，>8%买方市场，>15%恐慌性下跌 | 链家/贝壳内部数据、房管局网签价对比 | 周度 |

### 📍 法拍与不良资产指标（2项）

| 指标 | 为什么重要 | 数据获取渠道 | 更新频率 |
|------|-----------|-------------|---------|
| **法拍房数量** | 市场风险的先行指标；数量激增=断供潮/企业破产潮 | 阿里拍卖、京东拍卖、人民法院诉讼资产网 | 周度 |
| **法拍数据**（折扣率+流拍率+成交价） | 市场真实底价的发现机制；平均折扣率<70%=市场恐慌 | 阿里拍卖周报、京东拍卖周报、海豚选房APP | 周度 |

### 📍 货币与金融指标（5项）

| 指标 | 为什么重要 | 数据获取渠道 | 更新频率 |
|------|-----------|-------------|---------|
| **M2指数**（M2增速） | 货币供应量增速；>10%=宽松，<8%=紧缩 | 央行月度金融统计数据 | 月度 |
| **银行信贷额度变化** | 房贷放款速度、额度松紧；额度收紧=成交周期拉长 | 央行上海总部、各大银行信贷部、中介反馈 | 月度 |
| **居民杠杆率** | 居民债务/GDP；>60%=加杠杆空间有限，>70%=危险区间 | 央行、BIS国际清算银行、国家金融与发展实验室 | 季度 |
| **收入/杠杆比**（偿债收入比） | 居民债务/可支配收入；>40%=偿债压力过大 | 央行储户问卷调查、统计局居民收入数据 | 季度 |
| **通胀预期**（CPI+PPI） | 决定实际利率水平；通胀上行=实际利率下降=刺激买房 | 国家统计局CPI/PPI数据、央行储户通胀预期调查 | 月度 |

### 📍 情绪与微观指标（3项）

| 指标 | 为什么重要 | 数据获取渠道 | 更新频率 |
|------|-----------|-------------|---------|
| **挂牌量数据**（冰山指数） | 市场抛压的直接体现；**冰山指数**是二手房挂牌量最权威数据源 | [冰山指数官网](http://www.bingshanchart.com/)、链家APP前台数据 | 日/周 |
| **中介信心指数** | 一线从业者情绪；连续2月<50%=行业悲观 | 贝壳研究院经纪人信心指数、58同城经纪人调研 | 月度 |
| **VIP房源（急售）占比** | 业主心态的极致体现；>20%=下跌加速 | 绿中介APP筛选标签统计 | 周度 |

## 三、三层监控体系

### 【宏观层】决定3年趋势（人口+货币+政策）

| 类别 | 核心指标 | 关键阈值/观察点 |
|------|---------|----------------|
| **人口基本面** | 常住人口净流入/流出 | 连续两季度净流出>1万人=危险信号 |
| | 小学生入学人数 | 同比下跌>10%=学区房崩盘前兆 |
| **货币环境** | M2指数（M2增速） | >10%宽松，<8%紧缩 |
| | 通胀预期（CPI） | >3%=通胀压力，<1%=通缩风险 |
| | 银行信贷额度变化 | 放款周期>3个月=额度紧张 |
| | 居民杠杆率 | >70%=危险区间 |
| | 收入/杠杆比 | >40%=偿债压力过大 |
| **土地财政** | 土地流拍率 | >30%预示开发商集体看空 |
| **政策供给** | 保障房入市节奏 | 关注供给放量节奏 |

### 【中观层】决定6个月趋势（市场温度）

| 类别 | 核心指标 | 关键阈值/观察点 |
|------|---------|----------------|
| **价格指标** | 成交价 | 连续3月环比下跌=趋势转弱 |
| | 二手房议价空间 | <3%卖方市场，>8%买方市场，>15%恐慌 |
| **供需指标** | 库存量 | 新房>800万㎡、二手>20万套=高压 |
| | 库存去化周期 | <6个月上涨动力，>18个月下跌压力 |
| | 供需关系 | 供需比>1.5:1=买方市场 |
| **不良资产** | 法拍房数量 | 月增量>500套=风险上升 |
| | 法拍数据（折扣率） | <70%=市场恐慌 |
| **市场情绪** | 中介信心指数 | <50%=行业悲观 |

### 【微观层】决定1个月波动（实时交易行为）

| 类别 | 核心指标 | 关键阈值/观察点 |
|------|---------|----------------|
| **抛压监测** | 挂牌量数据（冰山指数） | 单日新增>500套持续一周=业主出逃 |
| | VIP房源(急售)占比 | >20%=下跌加速 |
| **情绪验证** | 带看量 | 连续3周下降=成交量即将萎缩 |

## 四、趋势分析四象限框架

> 不要单看一个指标，建立交叉验证矩阵：

| 维度 | 多头信号组合 | 空头信号组合 |
|------|-------------|-------------|
| **供需力量** | 人口流入 + 库存<6个月 + 供需比<1:1 | 人口流出 + 库存>18个月 + 供需比>1.5:1 |
| **金融杠杆** | M2>10% + 信贷额度宽松 + 杠杆率<60% | M2<8% + 信贷收紧 + 偿债收入比>40% |
| **市场情绪** | 带看量3周增长 + 议价空间收窄 + 挂牌量下降 | 挂牌暴增 + VIP房源>20% + 中介指数<30 |
| **价格验证** | 成交价企稳 + 法拍折扣率>80% + 租金回报上升 | 成交价连跌 + 法拍<70% + 恐慌抛售 |

## 六、上海市场特色观察

### 🏙️ 上海房地产市场特点

1. **数据透明度高**：冰山指数、链家数据相对完善
2. **刚需支撑强**：人口持续流入，改善型需求旺盛
3. **分化明显**：核心区域（内环、滨江）与远郊差异巨大
4. **学区房敏感**：入学政策变化对特定板块影响剧烈
5. **政策响应快**：一线城市，政策传导效率高

### 📊 重点关注区域

| 区域类型 | 代表板块 | 关注要点 |
|---------|---------|---------|
| **核心资产** | 黄浦、静安、徐汇内环 | 成交价稳定性、议价空间 |
| **改善热点** | 前滩、大虹桥、张江 | 库存去化周期、倒挂幅度 |
| **刚需板块** | 松江、嘉定、青浦新城 | 库存量、供需关系 |
| **学区房** | 各区内环老破小 | 入学人数变化、挂牌量数据 |
