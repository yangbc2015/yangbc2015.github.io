## 可视化项目

探索数据可视化的艺术与技术，将复杂数据转化为直观的视觉故事。

## 技术栈

### 前端可视化
- D3.js - 自定义交互图表
- ECharts - 商业级图表库
- Three.js - 3D 可视化

### 数据处理
- Python + Pandas - 数据清洗
- Jupyter Notebook - 探索性分析
- Apache ECharts - 报表生成

## 项目展示

### 1. 知识图谱
交互式网络图，展示想法之间的关联。

### 2. 实时数据大屏
WebSocket 实时数据可视化。

### 3. 3D 数据景观
使用 Three.js 创建沉浸式数据体验。

## 可视化原则

1. **清晰性** - 数据应该易于理解
2. **诚实性** - 准确呈现数据，不误导
3. **效率性** - 用最少墨水传递最多信息
4. **美观性** - 美观的设计更易被接受
